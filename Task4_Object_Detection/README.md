# YOLOv8 Object Detection System

A computer vision system that detects and localizes objects in images using deep learning.

---

## 🚀 Features
- Object detection using YOLOv8  
- Bounding box visualization  
- Adjustable confidence threshold  
- Save detection results  

---

## 🧠 Tech Stack
- Python  
- YOLOv8  
- OpenCV  
- Tkinter  

---

## 📌 Overview
This project integrates a deep learning-based object detection model with a graphical interface for visualizing detection results on images.
